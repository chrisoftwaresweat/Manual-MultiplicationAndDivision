# ➕ Java Manual Math (Recursion Calculator)

## 📌 Description
This program demonstrates how to perform **multiplication and division manually using recursion** in Java.

Instead of using built-in operators (`*` and `/`), the program:
- Uses **repeated addition** for multiplication
- Uses **repeated subtraction** for division

It also includes a simple **menu-driven system** for user interaction.

---

## 🎯 Features
- Manual multiplication (without `*`)
- Manual division (without `/`)
- Uses recursion for calculations
- Menu-based user interface
- Handles invalid input choices
- Prevents division by zero
- Allows repeated operations until exit

---

## 🧠 Concepts Used
- Recursion
- Methods
- Conditional statements (if-else, switch)
- Loops (do-while)
- User input (Scanner)
- Basic arithmetic logic (addition & subtraction)

---

## ▶️ Program Flow
1. Program displays a menu:
   - Manual Multiplication
   - Manual Division
   - Exit
2. User selects an option
3. Based on choice:
   - Multiplication → uses recursive addition
   - Division → uses recursive subtraction
4. Displays the result
5. Repeats until user exits

---

## 💡 Key Learning
- Multiplication can be done using repeated addition
- Division can be done using repeated subtraction
- Recursion solves problems by calling itself
- Always include a **base case** to avoid infinite recursion
- Input validation is important (e.g., division by zero)

---

## ⚠️ Note
- Division returns only the **quotient** (no remainder)
- If divided by 0, the program returns `-1`
- Designed for learning purposes (not optimized for large numbers)
